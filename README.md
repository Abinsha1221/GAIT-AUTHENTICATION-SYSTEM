# GAIT-AUTHENTICATION-SYSTEM

AI-powered gait authentication system using pose estimation and machine learning.

## Overview

This project is a real-time gait-based biometric authentication system developed as a final year B.Tech project.  
The system identifies and authenticates users based on their walking patterns using computer vision and machine learning techniques.

## Features

- Real-time gait authentication system
- Pose estimation using computer vision
- User registration and authentication
- Deep learning-based gait recognition
- Live video-based verification
- Responsive frontend using HTML, CSS, and JavaScript
- Flask backend integration
- Secure user gait data handling
